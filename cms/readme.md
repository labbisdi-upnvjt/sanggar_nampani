Halaman CMS : Untuk halaman admin untuk mengelola konten sanggar

Halaman ini tidak tampil di sisi client-side, melainkan hanya tampil pada admin melalui halaman login dan sistem harus logout jika admin sudah selesai melakukan aktivitas di halaman CMS

CMD berinteraksi langsung dengan backend dan frontend,CMS akan mengirimkan permintaan ke backend untuk menyimpan, mengupdate, menghapus dan mengambil data yang dibutuhkan untuk ditampilkan pada halaman CMS, lalu frontend akan menjawab melalui ApiController yang memiliki koneksi ke database
